# 베프(BEFF) 사용자 앱 백엔드 추가 필요 목록

이 문서는 현재 사용자 앱 프론트가 실제 서비스가 되기 위해 필요한 백엔드 작업을 정리한 것입니다. 현재 프론트는 모두 Mock 데이터를 사용합니다.

## 1. 인증/인가

- 학교 이메일 인증 (도메인 화이트리스트, OTP 코드 검증)
- 모바일 OTP 인증 (SOS·매칭 안전 흐름)
- 사용자 역할: 이용자(장애학생) / 제공자(비장애학생) / 학교 담당자
- 역할별 메뉴·기능 노출 제어
- 학교 협약(MOU) 여부에 따른 활동확인서 발급 자격 분기
- 신원 인증 배지 발급(학교 이메일 인증, 매너 점수)

## 2. 사용자/프로필

- 닉네임, 학교/학과, 자기소개 저장
- 접근 프로파일 저장 (필요한 도움 / 제공 가능한 도움)
- 친구 관계(매칭 후 친구 등록/해제) 저장
- 알림·접근성 환경설정 저장
- 차단/신고 목록 저장

## 3. 무장애 지도 / 시설

- 시설 CRUD (학교 담당자 입력 + 공공데이터 시드)
- 사용자 즐겨찾기 저장
- 사용자 시설 신고(고장/공사/추가 정보) 접수
- 시설 신뢰도 배지(BF 인증/사용자 인증/관리자 검증)
- 캠퍼스별 데이터 분리
- 지하철·외부 BF 시설 데이터 동기화 배치

## 4. 매칭

- 동행 요청/제공 CRUD
- 매칭 알고리즘 (접근 프로파일·시간·장소·과거 매칭 기록 기반)
- 매칭 단계 상태 머신 (접수→매칭→일정조율→동행→종료/설문)
- 자동 리마인더 (활동 24시간 전, 1시간 전)
- 활동 시작/종료 체크인
- 만족도·매너 평가 저장
- 매칭 취소·사유 저장

## 5. 봉사시간 / 활동확인서

- 활동 인정 시간 계산 (체크인-체크아웃 기준)
- 봉사시간 누적/학기 목표 저장
- 활동확인서 PDF 생성
- 학교 공동 명의 발급용 메타데이터(MOU 정보, 학교 인장)
- 학점 신청용 서류 자동 생성
- 발급 이력 저장

## 6. 긴급 도움 호출(SOS)

- SOS 호출 접수 (위치/메시지/대상 동시 전송)
- 호출 대상별 라우팅 (보호자 SMS/푸시, 지원센터 알림톡, 보안실 알림, 119 안내)
- 응답자 배정 및 응답 시간 기록
- 사후 처리/완료 사유 저장
- 호출 이력(개인/관리자) 저장
- 음성 메시지 첨부

## 7. 커뮤니티

- 게시판 카테고리(자유/정보/동행후기/질문) CRUD
- 게시글/댓글/좋아요
- 인기글 산정 배치
- 신고/숨김 처리

## 8. 알림

- 푸시 토큰 등록·갱신
- 알림 유형별 사용자 토글 반영
- 알림 발송 이력

## 9. DB 설계 후보

| 테이블 | 설명 |
|---|---|
| `users` | 사용자 계정 |
| `user_roles` | 사용자 역할 |
| `access_profiles` | 접근 프로파일(필요한 도움/제공 도움) |
| `friendships` | 친구 관계 |
| `campuses` | 학교/캠퍼스 |
| `facilities` | 시설(무장애 지도) |
| `facility_reports` | 시설 사용자 신고 |
| `match_categories` | 매칭 카테고리 |
| `matches` | 동행 매칭 |
| `match_applications` | 매칭 신청 |
| `match_reviews` | 매너 평가/회고 |
| `volunteer_records` | 봉사시간 기록 |
| `certificates` | 활동확인서 발급 이력 |
| `sos_calls` | SOS 호출 |
| `sos_targets` | 사용자별 호출 대상(보호자 등) |
| `posts` / `comments` | 커뮤니티 |
| `notifications` | 알림 |
| `audit_logs` | 변경/조회 감사 로그 |

## 10. 우선순위

| 우선순위 | 작업 |
|---|---|
| 1 | 인증(학교 이메일), 사용자/프로필 CRUD, 접근 프로파일 저장 |
| 2 | 매칭 CRUD + 상태 머신, 봉사시간 기록 |
| 3 | SOS 호출 접수·라우팅·이력 |
| 4 | 활동확인서 PDF 생성, 학교 공동 명의 |
| 5 | 시설 CRUD + 공공데이터 동기화 |
| 6 | 알림·푸시, 자동 리마인더 |
| 7 | 커뮤니티 |
